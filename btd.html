Brain Tumor Detection

Project With Christopher Kinder

Developed a deep learning pipeline for pixel-wise brain tumor segmentation on MRI scans using a multi-headed convolutional network in PyTorch, based on the Hugging Face dataset dwb2023/brain-tumor-image-dataset-semantic-segmentation. Implemented training, evaluation, and visualization workflows, along with GitHub Actions for linting and formatting to ensure code quality. We achieved an accuracy of 0.995 for the tumor classification and a IoU of 0.53 for the bounding box segmentation.
